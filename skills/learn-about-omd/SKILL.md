# Learn About OMD

Analyze Oh-My-Droid usage patterns and provide personalized recommendations.

## What This Does

1. **Analyzes** command usage history
2. **Identifies** patterns in how you work
3. **Recommends** features you might not be using
4. **Suggests** optimizations for your workflow

## Insights Provided

- Most used commands and modes
- Agent delegation patterns
- Token usage by mode
- Underutilized features
- Workflow optimization tips

## Analysis Areas

### Command Usage
- Which commands do you use most?
- Which modes are you not using?

### Agent Patterns
- Which droids get spawned most?
- Are you using the right tier for tasks?

### Efficiency
- Are you using the ship workflow effectively?
- Are you using parallelism effectively?

### Recommendations
Based on your patterns:
- Suggest new workflows
- Highlight underused features
- Optimize for your use case

## Output Format

```
## OMD Usage Analysis

### Your Patterns
- Most used: /omd-autopilot (45%)
- Least used: /omd-swarm (2%)

### Recommendations
1. Use /omd-ship for complex tasks
2. Consider /omd-pipeline for reviews
3. Use /omd-ralplan when planning needs extra rigor

### Optimization Tips
- You could save tokens by using LOW tier for searches
- Your workflow would benefit from /omd-ralplan
```
