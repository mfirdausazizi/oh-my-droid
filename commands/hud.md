# HUD - Status Display

Show current Oh-My-Droid status and active operations.

## Display

1. **Active Mode**
   - Current mode (normal, autopilot, ralph, ultrawork, swarm)
   - Iteration count if in ralph mode

2. **Todo Progress**
   - Pending tasks
   - In-progress tasks
   - Completed tasks

3. **Background Agents**
   - Number of active background tasks
   - Agent types running

4. **State Files**
   - ralph-state.json status
   - ultrawork-state.json status
   - swarm.db status

5. **Memory**
   - Personal memories count
   - Project memories count
