# Learn About OMD

Analyze your Oh-My-Droid usage patterns and get personalized recommendations.

## What This Does

1. **Analyzes** your command usage history
2. **Identifies** patterns in how you work
3. **Recommends** features you might not be using
4. **Suggests** optimizations for your workflow

## Insights Provided

- Most used commands and modes
- Agent delegation patterns
- Token usage by mode
- Underutilized features
- Workflow optimization tips

## Usage

Just run `/omd-learn-about-omd` and I'll analyze your patterns.
